# رادار الزبايل

تطبيق محلي للماك والوندوز يحتفظ بقائمة Riot IDs وينبهك إذا دخل أحدهم معك أو ضدك في مباراة League of Legends.

## التحميل

نزّل آخر نسخة من صفحة [Releases](https://github.com/othmanalomair/redar/releases/latest):

- **Mac (Apple Silicon):** ملف `.dmg`
- **Windows x64:** ملف `Setup.exe` للتثبيت أو ملف Portable للتشغيل المباشر

على الماك، إذا منع النظام فتح التطبيق أول مرة، اضغط عليه بالزر اليمين واختر **Open**.

## الاستخدام

1. أضف اللاعب باستخدام Riot ID كاملًا، مثل `PlayerName#TAG`.
2. اترك التطبيق يعمل في Menu Bar على الماك أو System Tray على Windows.
3. عند بدء مباراة، يفحص التطبيق قائمة اللاعبين ويعرض إشعارًا إذا وجد تطابقًا.

يظهر التنبيه مرة واحدة في المباراة كإشعار للنظام وكشريط واضح فوق اللعبة لمدة 8 ثوانٍ. تقدر توقف تنبيه الشاشة أو تجرب شكله من الإعدادات.

القائمة محفوظة محليًا على جهازك ولا يتم إرسالها إلى أي سيرفر.

## التشغيل للتطوير

```bash
npm install
npm start
```

الإغلاق العادي يخفي النافذة ويترك التطبيق في Menu Bar أو System Tray. اختر «خروج نهائي» من الأيقونة لإطفائه.

## إخراج نسخة تثبيت

```bash
npm run dist:mac
npm run dist:win
```

يقرأ التطبيق قائمة اللاعبين محليًا أثناء المباراة من Live Client Data API على المنفذ `2999`. لا يطلب كلمة مرور Riot ولا يرسل قائمتك إلى أي سيرفر.
